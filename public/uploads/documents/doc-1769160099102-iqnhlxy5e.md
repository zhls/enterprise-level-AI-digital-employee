# AI占卜师 - 神秘学风格数字人

基于魔珐星云3D数字人技术的AI占卜师，支持自定义知识库和RAG向量检索。

## 功能特性

- 🔮 塔罗牌占卜（单张/三张/凯尔特十字牌阵）
- 📚 自定义知识库上传（txt/md/pdf）
- 🧠 RAG向量检索增强解读
- 🤖 3D数字人驱动
- 🎨 神秘学风格UI

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 前端: http://localhost:5173
# 后端: http://localhost:5176
```

## 技术栈

- **前端**: React + Vite + TypeScript + TailwindCSS
- **后端**: Express + TypeScript
- **AI服务**: 魔搭社区 (Qwen3-VL + Qwen3-Embedding-8B)
- **数字人**: 魔珐星云具身驱动SDK
